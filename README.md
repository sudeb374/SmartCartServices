# \# SmartCart - Customer Segmentation using Unsupervised Machine Learning



### \## Project Overview



SmartCart is an Unsupervised Machine Learning project focused on analyzing customer purchasing behavior to discover hidden patterns and segment customers into meaningful groups.



The objective of this project is to:

\- Identify different customer segments

\- Understand purchasing behavior

\- Improve targeted marketing strategies

\- Enable personalized shopping experiences



### \## Machine Learning Techniques Used



\- K-Means Clustering

\- Agglomerative clustering

\- PCA (Principal Component Analysis)

\- Data Scaling , Preprocessing and Encoding 



### \## Dataset Description



The dataset contains customer shopping behavior information such as:

1.Customer Demographics

&nbsp;	- CustomerID – Unique customer identifier

&nbsp;	- Age – Customer age

&nbsp;	- Marital status

&nbsp;	- Education

&nbsp;	- Income

2.Purchase Behaviour

&nbsp;	-MntWines : Amount spnet on wine products

&nbsp;	-MntFruits : Amount spnet on fruits

&nbsp;	-MntMeatProducts

&nbsp;	-MntFishProducts

3.Purchase Behaviour

&nbsp;	-NumDealsPurchases : Purchases made using discounts

&nbsp;	-NumWebPurchases : Purchases made through website

&nbsp;	-NumStorePurchases : Purchases made in physical stores

&nbsp;	-NumWebVisitsMonth : Number of visits to website per month

4.Customer feedback

&nbsp;	-Recency : Number of days since last purchase 

&nbsp;	-Complain : Customer complained in last 2 years(1=Yes,0=No)



⚠️ Note: The full dataset is not included in this repository due to size/privacy considerations.



To use this project:

1\. Download this type of dataset.

2\. Place it inside a `data/` folder.

3\. Update the file path in the notebook if needed.
